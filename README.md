# dotfiles

### Personal Arch Linux configuration files - WIP

My personal dotfiles for a Hyprland-based desktop on Arch Linux. Colors are generated dynamically using [matugen](https://github.com/InioX/matugen), feeding into GTK 3/4, Qt, Waybar, hyprlock, an RGB LED strip, ...

> **Note:** These dotfiles are for personal use. Feel free to take inspiration, but don't expect everything to work out of the box
